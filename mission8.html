Port Security

Profil Projets Veille Parcours

🎯 Objectif

Sécuriser les ports des switchs en limitant à une seule adresse MAC par port grâce à Port Security. Toute violation entraîne la désactivation automatique du port.

🧠 Principe

  • 1 seule MAC autorisée par port
  • Apprentissage automatique (sticky)
  • Violation → shutdown du port
  • Protection contre MAC flooding et intrusions physiques

🛠️ Configuration de base

interface fa0/1 switchport mode access switchport port-security switchport port-security maximum 1 switchport port-security violation shutdown switchport port-security mac-address sticky exit

🔁 Configuration sur plusieurs ports

interface range fa0/1 - 10 switchport mode access switchport port-security switchport port-security maximum 1 switchport port-security violation shutdown switchport port-security mac-address sticky exit

🔍 Vérifications

État du port

show port-security interface fa0/1

✔ Maximum MAC = 1 | Violation = Shutdown

MAC apprises

show port-security address

🚨 Comportement en cas de violation

  • Port passe en err-disabled
  • Plus de trafic
  • LED rouge (Packet Tracer)

🧪 Test

  • PC connecté → OK
  • Remplacement PC → violation
  • Port désactivé automatiquement

🔧 Réactivation port

interface fa0/1 shutdown no shutdown

✅ Résultat attendu

  • 1 seule MAC par port
  • Ports sécurisés
  • Blocage automatique des intrusions

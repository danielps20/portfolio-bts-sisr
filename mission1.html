⚙️ 2. Création de l’infrastructure

Équipements

  • 1 Router 2911
  • 4 Switch 2960
  • 2 PC
  • 1 Laptop
  • 4 serveurs
  • 1 Access Point

Connexions

  • PC → Switch
  • Switch → Switch
  • Switch → Serveur
  • Switch → Access Point
  • Laptop → Wi-Fi
  • Routeur relie les 3 réseaux

🌐 3. Configuration du routeur

Configuration CLI :

interface gig0/0 ip address 172.17.0.1 255.255.255.0 no shutdown interface gig0/1 ip address 10.0.0.1 255.255.255.0 no shutdown interface gig0/2 ip address 193.253.0.1 255.255.255.0 no shutdown

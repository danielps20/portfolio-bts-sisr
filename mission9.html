🎯 Objectif

Mettre en place un réseau IPv6 avec routage statique et autoconfiguration SLAAC entre deux routeurs.

🧠 Topologie

  • R1 ↔ R2 (FC00::/64)
  • LAN R1 : 2001:DB8:1:1::/64
  • LAN R1 : 2001:DB8:1:2::/64
  • LAN R2 : 2001:DB8:2::/64

⚙️ Configuration R1

enable conf t ipv6 unicast-routing interface fa0/0 ipv6 address FC00::1/64 no shutdown interface fa0/1 ipv6 address 2001:DB8:1:1::1/64 no shutdown interface fa1/0 ipv6 address 2001:DB8:1:2::1/64 no shutdown ipv6 route 2001:DB8:2::/64 FC00::2 end

⚙️ Configuration R2

enable conf t ipv6 unicast-routing interface fa0/0 ipv6 address FC00::2/64 no shutdown interface fa0/1 ipv6 address 2001:DB8:2::1/64 no shutdown ipv6 route 2001:DB8:1:1::/64 FC00::1 ipv6 route 2001:DB8:1:2::/64 FC00::1 end

🖥️ Configuration des postes

PC1

  • IPv6 : 2001:DB8:1:1::10/64
  • Passerelle : 2001:DB8:1:1::1

PC2

  • IPv6 : 2001:DB8:1:2::10/64
  • Passerelle : 2001:DB8:1:2::1

PC0

  • IPv6 : 2001:DB8:2::10/64
  • Passerelle : 2001:DB8:2::1

🔍 Vérifications

show ipv6 interface brief show ipv6 route

Tests : ping entre réseaux IPv6

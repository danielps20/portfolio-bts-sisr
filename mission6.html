🎯 Objectif

Mise en place de VLAN pour séparer deux ligues sportives sans routage inter-VLAN.

🧩 Plan d’adressage

VLAN Nom Réseau
2 ESCRIME 172.17.2.0/24
3 VOLLEY 172.17.3.0/24

🔧 Création des VLAN

enable configure terminal vlan 2 name ESCRIME exit vlan 3 name VOLLEY exit

🔌 Affectation des ports

VLAN 2 - ESCRIME

interface fa0/1 switchport mode access switchport access vlan 2 exit

VLAN 3 - VOLLEY

interface fa0/3 switchport mode access switchport access vlan 3 exit

🔗 Trunk entre switches

interface fa0/1 switchport mode trunk switchport trunk allowed vlan 2,3 exit interface fa0/2 switchport mode trunk switchport trunk allowed vlan 2,3 exit

💻 Configuration des postes

ESCRIME

  • IP : 172.17.2.X
  • Masque : 255.255.255.0
  • Passerelle : aucune

VOLLEY

  • IP : 172.17.3.X
  • Masque : 255.255.255.0
  • Passerelle : aucune

🧪 Résultat attendu

  • Séparation des VLAN OK
  • Communication intra-VLAN OK
  • Inter-VLAN bloqué (pas de routage)

🎯 Objectif

Mettre en place une supervision centralisée des équipements réseau via EyesOfNetwork (basé sur Nagios). Surveillance des serveurs, routeurs, switchs, services et périphériques.

🧠 Contexte

  • Outil : EyesOfNetwork 3.1
  • Basé sur Nagios (ITIL)
  • Protocoles : SNMP / plugins Nagios
  • Port SNMP : UDP 161 / 162

⚙️ Activation SNMP

Cisco (Switch / Routeur)

snmp-server community NomCommunauté

Windows

  • Activer SNMP via fonctionnalités Windows
  • Configurer communauté SNMP

🔍 Test SNMP

  • Interface : Administration → snmpwalk
  • Tester une IP d’équipement

🧩 Configuration Nagios

Modifier communauté SNMP

$USER2$ → ex : infosio

Contact admin

Modifier email dans configuration Nagios

🖥️ Ajout d’hôte

  • Administration → Equipements → Add Host
  • Template : GENERIC_HOST
  • Exporter configuration après ajout

📊 Vérification état

Disponibilités → Evénements → Vue équipements

Si PENDING → valider commande

🧪 Commandes utiles SNMP

CPU Load

./check_snmp_load.pl -H IP -C comm -w 80 -c 90

Interfaces réseau

./check_snmp_int.pl -H IP -C comm -n interface -v

Windows DNS

./check_snmp_win.pl -H IP -C comm -n DNS

DHCP

./check_dhcp_addfree -H IP -C comm -w 50 -c 10

Imprimante

./check_snmp_printer -H IP -C comm -x MODEL

🎯 Objectif

Mettre en place un serveur Syslog permettant de centraliser les journaux (logs) des équipements réseau : routeurs, switchs, serveurs et pare-feu.

🟢 Option 1 - Linux (recommandé)

1️⃣ Installation

sudo apt update sudo apt install rsyslog -y

2️⃣ Activation réception logs

# UDP module(load="imudp") input(type="imudp" port="514") # TCP module(load="imtcp") input(type="imtcp" port="514")

3️⃣ Redémarrage

sudo systemctl restart rsyslog

4️⃣ Vérification logs

tail -f /var/log/syslog

5️⃣ Pare-feu

sudo ufw allow 514/udp sudo ufw allow 514/tcp

🪟 Option 2 - Windows

Outils

  • Kiwi Syslog Server
  • Syslog Watcher
  • Graylog

Kiwi Syslog

  • Écoute UDP 514
  • Interface graphique pour logs
  • Configuration simple

⚙️ Configuration équipements réseau

conf t logging host 192.168.1.10 logging trap informational logging on exit

192.168.1.10 = IP du serveur Syslog

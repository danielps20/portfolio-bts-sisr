🧩 Objectif

Mettre en place un EtherChannel entre Switch4 et Switch5 afin d’augmenter la bande passante et assurer la redondance.

🔎 Topologie

  • Switch4 ↔ Switch5
  • Liaisons physiques : Fa0/2 et Fa0/4
  • Protocol : LACP
  • VLAN : 2 et 3

🔧 Configuration Switch5

enable configure terminal interface range fa0/2 , fa0/4 switchport mode trunk switchport trunk allowed vlan 2,3 channel-group 1 mode active exit interface port-channel 1 switchport mode trunk switchport trunk allowed vlan 2,3 exit end

🔧 Configuration Switch4

enable configure terminal interface range fa0/1 , fa0/4 switchport mode trunk switchport trunk allowed vlan 2,3 channel-group 1 mode active exit interface port-channel 1 switchport mode trunk switchport trunk allowed vlan 2,3 exit end

🔍 Vérifications

EtherChannel

show etherchannel summary

✔ SU = actif | P = ports agrégés

Trunk

show interfaces trunk

✔ Port-channel1 en trunk VLAN 2,3

🚀 Résultat attendu

  • Bande passante doublée
  • Redondance automatique
  • VLAN 2 et 3 fonctionnels
  • STP optimisé

🧪 Test

  • Ping entre VLAN 2 et 3
  • Débrancher un câble → trafic toujours OK
